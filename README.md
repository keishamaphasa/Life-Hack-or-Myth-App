# Life-Hack-or-Myth-App
 A native Android flashcard quiz app built with Kotlin that challenges users to distinguish between real life hacks and urban myths.
 
#  Purpose
This app helps users test their common sense and learn useful real-world 
shortcuts by identifying genuine life hacks from common urban myths.

---

## Features
- Welcome screen with app description
- 10 flashcard questions (Hack or Myth)
- Instant feedback after each answer
- Final score with personalised feedback
- Review screen showing all correct answers and explanations

## How to Use
1. Press **Start** on the welcome screen
2. Read each statement and select **Hack** or **Myth**
3. Get instant feedback on your answer
4. View your final score and personalised message
5. Press **Review Answers** to see all correct answers and explanations
6. Press **Reset** to play again
